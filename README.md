# IshikaChatBot ✨

Advanced Telegram Group Manager Bot powered by Pyrogram + MongoDB

## Features
- Admin Tools
- Notes, Filters, Welcome
- AFK, Couples, Tagall
- Auto Read
- MongoDB Storage

## Deploy on Render
1. Set `PYTHON_VERSION=3.11.9`
2. Set `Start Command = python -m ishikachat`
3. Add all ENV vars from.env.example
